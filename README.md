
# 飞花令古风诗词游戏

一个基于Vue.js和Coze AI的古风诗词对战游戏。

## 功能特点

- 🎮 三种难度模式：入门级、进阶级、困难级
- 🤖 AI智能对手：基于Coze AI的诗词生成
- 🎵 背景音乐：不同难度对应不同古风音乐
- 📚 诗词背景：AI生成的诗句背景故事
- ⏰ 计时系统：根据难度设置不同时间限制
- 🎯 回合制：支持跳过回合和认输功能

## 技术栈

- 前端：Vue.js + Tailwind CSS
- 后端：Node.js + Express
- AI：Coze API
- 音乐：HTML5 Audio API

## 安装和运行

### 环境要求

- Node.js 18+ 
- npm 或 yarn

### 安装步骤

1. 解压项目文件到任意目录
2. 双击 `启动游戏.bat` 启动游戏
3. 打开浏览器访问：http://localhost:3000

**或者手动启动：**
```bash
npm install
node server.js
```

### 配置说明

✅ **Coze API 已预配置完成**，无需修改
✅ **音乐文件已包含**，无需额外配置

## 游戏规则

1. 选择难度开始游戏
2. 根据关键字说出包含该字的古诗词
3. 与AI轮流对诗
4. 完成指定回合数即可获胜

## 注意事项

- 确保网络连接正常（需要调用Coze API）
- 音乐文件需要放在项目根目录
- 建议使用现代浏览器（Chrome、Firefox、Edge等）

## 故障排除

如果遇到问题：

1. 检查Node.js版本是否为18+
2. 确认所有依赖已正确安装
3. 检查Coze API配置是否正确
4. 查看控制台错误信息
