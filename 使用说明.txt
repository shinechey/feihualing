飞花令古风诗词游戏 - 使用说明
=====================================

🎮 游戏介绍
-----------
这是一款基于Vue.js和Coze AI的古风诗词对战游戏，玩家可以与AI进行飞花令对决。

🚀 快速开始
-----------
1. 确保已安装 Node.js 18+ 版本
2. 双击 "启动游戏.bat" 文件
3. 等待服务器启动完成
4. 打开浏览器访问 http://localhost:3000

⚙️ 配置要求
-----------
✅ Coze API 已配置完成，无需修改
✅ 音乐文件已包含，无需额外配置

📁 项目文件
-----------
- index.html              # 游戏前端页面
- server.js               # 后端服务器（API已配置）
- package.json            # 项目依赖配置
- 启动游戏.bat            # 一键启动脚本
- README.md               # 详细说明文档
- 使用说明.txt            # 快速使用指南
- 音乐文件/               # 背景音乐（4个MP3文件）

🎵 音乐文件
-----------
✅ 音乐文件已包含在项目包中
✅ 包含4个背景音乐文件，无需额外配置

🔧 故障排除
-----------
- 如果启动失败，请检查 Node.js 版本
- 如果音乐无法播放，请检查音乐文件
- 如果AI无法响应，请检查API配置
- 详细问题请查看 README.md 和 配置说明.md

📞 技术支持
-----------
如有问题，请参考项目内的说明文档，或联系开发者。

祝您游戏愉快！🎉
